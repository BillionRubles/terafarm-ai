Why this architecture matters

Memory is the missing layer
Every AI agent today has isolated context. Terafarm provides a single, permissioned memory that any agent can read or write to.
MCP-native orchestration
Built around the Model Context Protocol so any agent or tool can plug into the same memory and routing fabric.
Privacy by design
Every read/write to memory passes through the privacy layer. Per-agent permissions, encryption, and full audit log.
Business Case
Why Terafarm.ai now.
Hackathon Track · Build — Net-New Agents
Terafarm.ai is the shared memory layer
for AI agents.
As people use more AI systems, context becomes fragmented across every tool, model, and device. Terafarm connects AI assistants, tools, devices and automations into one memory network — so every agent in your life works with full context, not from zero.

The Problem

Fragmented context
The average user touches 5+ AI tools weekly. None of them remember each other. Every conversation starts from scratch.
Time wasted re-explaining
Users repeat themselves across ChatGPT, Claude, Gemini, Notion, Copilot — losing hours per week to context-rebuilding.
No agent-to-agent coordination
Agents can't hand off tasks or share state. Multi-step workflows across tools require manual stitching by the user.

The Solution

One shared memory
A permissioned, encrypted memory layer that every AI agent can read and write through standardised protocols.
Multi-agent orchestration
One user command routes to many specialist agents in parallel — Travel, Finance, Schedule, Car, Assistant — each writing to shared memory.
Single interface, every AI
Premium dashboard for the AI ecosystem. Apple × Tesla × Jarvis feel. Mobile-first, desktop-ready.

Tech Stack
Production-ready foundations for an AI OS.

Frontend

Framework
Vanilla + React (planned)
MVP is dependency-free for max portability; production target is React 18 + Next.js 14.
Styling
CSS + Tailwind (planned)
Custom design tokens, glassmorphism cards, cold-tech palette. Production migrates to Tailwind.
Typography
Rajdhani + Exo 2
Display + body pairing via Google Fonts. Distinct, futuristic, readable.
Hosting
Vercel · GitHub
Static deploy from Git. Zero-config preview deployments, edge network.
Mobile
PWA → React Native
Installable PWA first; native shell on iOS/Android using the same component library.
Voice
Web Speech API
Voice command capture in-browser; falls back to OS voice on native.

Agent Orchestration

Protocol
MCP (Model Context Protocol)
Standard interface for agents and tools to register, discover, and call each other.
Router
LLM-based intent classifier
Routes natural language to one or many agents using Gemini / Claude / GPT-4.
Function calling
OpenAI · Anthropic · Google
Each connected model uses its native tool-use interface, normalised by Terafarm.

Shared Memory

Vector store
Pinecone / pgvector
Semantic memory: embedding-based recall with metadata filtering.
Object store
Supabase / Postgres
Structured memory: events, tasks, preferences, audit log.
Sync
CRDTs · Real-time channels
Conflict-free updates across devices and agents.

Cloud + Security

Cloud
Google Cloud · Vertex AI
Compute, Gemini API, embeddings, secret manager.
Auth
OAuth 2.0 + WebAuthn
Social sign-in, passkeys, optional 2FA.
Encryption
AES-256 · TLS 1.3
At-rest and in-transit. Per-agent access control lists.
